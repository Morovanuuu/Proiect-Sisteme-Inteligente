# Clasificarea Succesului Canalelor YouTube (1M Global Stats)

## 1. Sursa Datelor

- **Dataset:** [YouTube 1M Global Creator Analytics](https://www.kaggle.com/datasets/ehsanzx/youtube-1m-global-creator-analytics)
- **Data descarcarii:** 20 Martie 2026
- **Descriere:** Un set de date masiv ce contine statistici detaliate despre aproximativ 1 milion de creatori de continut de pe YouTube, incluzand categorii, numar de abonati, vizualizari si scoruri de sentiment.

## 2. Obiectivul Proiectului

Scopul acestui proiect este dezvoltarea unui **sistem inteligent de clasificare** care sa determine nivelul de succes si engagement al unui canal YouTube.

**Problema rezolvata:** Incercarea de a prezice categoria de succes a unui canal (ex: High/Medium/Low Engagement) in functie de variabile precum nisa continutului, frecventa postarilor si reactia publicului (sentiment score).

## 3. Tehnologii Utilizate

- **Limbaj de programare:** Python 3.12
- **Analiza si procesare date:** `pandas`, `numpy`
- **Vizualizare date:** `matplotlib`, `seaborn`
- **Machine Learning:** `scikit-learn` (urmeaza sa fie implementat)
- **Mediu de lucru:** Visual Studio Code & Jupyter Notebooks

## 4. Structura Depozitului

- `analiza_youtube.ipynb`: Notebook-ul principal ce contine incarcarea datelor, curatarea si analiza exploratorie (EDA).
- `README.md`: Documentatia proiectului.
- `.gitignore`: Fisier pentru excluderea fisierelor mari (CSV).
